# 🛡️ Cyber Security Internship – Task 2: Phishing Email Analysis

## ✅ Objective:
Analyze a suspicious phishing email and identify all potential phishing indicators to understand how attackers manipulate users.

---

## 🔧 Tools Used:
- Sample phishing email (text file)
- Manual analysis
- [MXToolbox Email Header Analyzer](https://mxtoolbox.com/EmailHeaders.aspx) (optional)

---

## 📊 Summary of Analysis:

- Spoofed sender email (`paypall.com` instead of `paypal.com`)
- Urgent/Threatening language used to trigger action
- Suspicious URL that doesn’t match the official domain
- No HTTPS in the URL
- Generic greeting (“Dear Customer”)
- Minor grammar/spelling issues
- No personalization
- Poor branding (missing official layout, logo, etc.)

---

## 🧠 Key Concepts Learned:

- How to identify spoofed domains and URLs
- Psychological tricks in phishing (urgency, fear, authority)
- Basic phishing indicators like generic greeting, fake links
- Manual phishing analysis and email forensics basics

---

## 🎯 Outcome:
Developed awareness of phishing techniques and sharpened email threat analysis skills. This task enhances readiness for identifying real-world phishing attacks.
